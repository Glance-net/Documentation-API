Get cobrowse selectors

Get cobrowse selectors for {setting} configuration and a particular group and site.

Required Permissions:

  • GROUP_OWNER role
Authorizations:
BearerAuth
path Parameters
group_id
required
integer

The group ID (Must be a positive unsigned integer)

param
required
string
Enum: "staging" "production"

The site environment (staging or production)

setting
required
string
Enum: "maskingelements" "maskingpages" "bookmarklinks" "docsharelinks" "allowedsites" "selectiveediting"

Cobrowse selector section like docsharelink, bookmarklink, maskedpage, maskingelements, allowedsites, selectiveediting

Responses

Glance API Server

https://www.glance.net/api/v1/account/groups/{group_id}/settings/cobrowse/{param}/{setting}

Create cobrowse selectors

Create an element for a specific setting category for the group. Supported settings include ALLOWINGSITES, DOCSHARELINKS, BOOKMARKLINKS, MASKINGPAGES, MASKINGELEMENTS, SELECTIVEEDITING.

Required Permissions:

  • GROUP_OWNER role
Authorizations:
BearerAuth
path Parameters
group_id
required
integer

The group ID (Must be a positive unsigned integer)

param
required
string
Enum: "staging" "production"

The site environment (staging or production)

setting
required
string
Enum: "maskingelements" "maskingpages" "bookmarklinks" "docsharelinks" "allowedsites" "selectiveediting"

ALLOWINGSITES, DOCSHARELINKS, BOOKMARKLINKS, MASKINGPAGES, MASKINGELEMENTS, SELECTIVEEDITING

Request Body schema: application/json
required
Array
id
integer

The primary key (masking_id) generated by the database when a selector is created. Required for update operations.

value
required
string

The selector value (e.g., URL for allowedsites, CSS selector for maskingelements like ".credit-card-number", ".ssn-field")

roles
Array of integers

Array of role IDs associated with this selector

directoryId
integer

Directory ID for foldered selectors (bookmarklinks, docsharelinks)

title
string

Title for the selector

modified
string <date-time>

Last modification timestamp

Publish

Publish cobrowse selectors from staging to production

Replicates the exact state of staging to production. Adds selectors that exist in staging but not in production, and deletes selectors that exist in production but not in staging.

Required Permissions:

  • GROUP_OWNER role
Authorizations:
BearerAuth
path Parameters
group_id
required
integer

The group ID (Must be a positive unsigned integer)

param
required
string

The setting to publish like selectiveediting, maskedpages, etc

Responses

Glance API Server

https://www.glance.net/api/v1/account/groups/{group_id}/settings/cobrowse/{param}/publish

Response samples

  • 200
  • 400
  • 500
Content type
application/json
{
  • "message": "Selectors published successfully"
}
